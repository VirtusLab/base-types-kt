# base-types-kt

Set of libraries with base types for Kotlin

### Currently supported types
- Railway-Oriented Programming with Project Reactor
    - MonoResult
    - FluxResult

### Coming in the future
- ROP with RxJava 2
    - SingleResult
    - CompletableResult
    - ObservableResult
- ROP with arrow-kt's IO
    - AsyncResult
- Refined data types
    - Digit
    - WholeNumber
    - ...
